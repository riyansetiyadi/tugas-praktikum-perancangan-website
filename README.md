# Tugas Praktikum Perancangan Website

Repositori ini berisi kode hasil pengerjaan tugas praktikum perancangan website. Fokus utama dari tugas ini adalah memahami penerapan teknik pengodean dokumen HTML, penggunaan tag semantik, serta implementasi selektor CSS (*inline* dan *internal*).

Halaman website dapat diakses melalui tautan berikut:
[https://riyansetiyadi.github.io/tugas-praktikum-perancangan-website/](https://riyansetiyadi.github.io/tugas-praktikum-perancangan-website/)

## 📝 Daftar Instruksi / Soal Tugas

Berikut adalah daftar instruksi tugas yang diimplementasikan langsung di dalam kode:

1. **Menerapkan *Inline CSS***
   - Membuat menu navigasi (`<nav>`).
   - Menyisipkan *style* langsung ke dalam tag (*Inline CSS*) untuk mengubah warna latar belakang navigasi menjadi `lightblue`.

2. **Menerapkan *Internal CSS* dengan *Class***
   - Membuat elemen `<section>`.
   - Membuat *class* CSS bernama `.background-pink` di dalam tag `<style>` pada bagian `<head>`, lalu menerapkannya pada *section* tersebut.

3. **Mengubah Ukuran *Font* Spesifik**
   - Mengatur ukuran *font* khusus untuk seluruh elemen `<h2>` menjadi `36px` melalui selektor tag di dalam *Internal CSS*.

4. **Menerapkan Struktur Bersarang (Latihan *Emmet*)**
   - Membuat struktur tag `<div>` baru.
   - Di dalam *div* tersebut, menambahkan tag `<h1>` berisi teks "Title" dan tag `<p>` bersarang yang berisi teks sepanjang 10 kata (*lorem ipsum*).
